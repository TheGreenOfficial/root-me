**Now lets understand what's SQL injection - Numeric is: Numeric SQL injection is an attack where an attacker manipulates number-based input fields (like IDs or quantities) to alter an SQL query and gain unauthorized access or control over a database. Because the backend query is almost certainly like this:**

```
SELECT title, content, date FROM news WHERE id = 2
```

**news_id is used as an integer, not as a string.**
<br><br>

