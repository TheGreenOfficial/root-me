**Now let's understand whats verb tampering lets take a example with following self explaned example php code:**

```
<?php
session_start();

$valid_user = "admin";
$valid_pass = "admin123";

if ($_SERVER["REQUEST_METHOD"] === "GET") {
    $user = $_POST["username"] ?? "";
    $pass = $_POST["password"] ?? "";

    if ($user === $valid_user && $pass === $valid_pass) {
       echo "<script>alert('login successo!');</script>";
    }
} else {
    echo "login failed";
    die();
}

<!DOCTYPE html>
```

**In that code there is verb tempering vuln cuz its only checking if request method is get and if yes its checking user pass and if incorrect it exits in else statement but if request method is somehtin else then we will get access to site. eg:**


**Original:**

``` GET /vuln.php?username=user&password=pass ```

``` ASDF /vuln.php?username=user&password=pass ```
